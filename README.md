# Crawlee Project

This project is a web scraping project using Node.js and Crawlee.

## Project Purpose
- Crawl websites to extract data
- Save scraped data into JSON/CSV

## Technologies
- Node.js
- Crawlee
- Cheerio (for HTML parsing)

## Folder Structure
```plaintext
Crawlee/
├── spiders/    ← crawler scripts
├── data/       ← scraped data
├── package.json
├── package-lock.json
└── README.md

## Setup
1. Clone the repo
2. Run `npm install` to install dependencies
3. Run your crawler using `node spiders/example.js`
